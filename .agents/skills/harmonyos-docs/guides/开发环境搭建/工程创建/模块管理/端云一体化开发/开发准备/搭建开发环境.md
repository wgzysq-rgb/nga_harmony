# 搭建开发环境

 基于Cloud Foundation Kit（云开发服务）的端云一体化开发需要使用DevEco Studio NEXT Developer Beta1及以上版本，如果您尚未安装或者安装版本过低，请[下载并安装最新版本的DevEco Studio](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-software-install)。

  端云一体化开发工程在初始化时需要从npm仓库下载依赖，如果需要代理才能访问网络，请[配置NPM代理](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-environment-config#section197296441787)。
