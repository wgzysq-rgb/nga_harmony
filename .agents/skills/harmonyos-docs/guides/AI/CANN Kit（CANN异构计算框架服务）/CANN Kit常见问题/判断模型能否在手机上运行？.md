# 判断模型能否在手机上运行？

 通过调用接口[HMS_HiAICompatibility_CheckFromFile](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/cannkit#hms_hiaicompatibility_checkfromfile)或者[HMS_HiAICompatibility_CheckFromBuffer](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/cannkit#hms_hiaicompatibility_checkfrombuffer)，传入编译后的模型文件或者模型buffer，如果返回“HIAI_COMPATIBILITY_COMPATIBLE”表示兼容性检查通过，模型可以在手机上运行。
