## 使用方法

1. 根据下方分类表，确定目标指南所属分类
2. 读取对应分类的子索引文件（路径：`guides/_index/编号.md`）
3. 在子索引中查找指南名称，获取文档相对路径
4. 拼接完整路径读取文档：`技能根目录/相对路径`（技能根目录 = SKILL.md 所在目录）
