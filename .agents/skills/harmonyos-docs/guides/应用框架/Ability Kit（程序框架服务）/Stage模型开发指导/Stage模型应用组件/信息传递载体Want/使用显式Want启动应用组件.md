# 使用显式Want启动应用组件

 在应用使用场景中，当用户在应用内点击某个按钮时，经常需要拉起指定UIAbility组件来完成某些特定任务。在启动UIAbility时，指定了abilityName和bundleName参数，可以使用显式Want方式启动UIAbility。

 针对应用的特定任务，用户需要通过点击应用内的按钮来启动指定的UIAbility组件。在启动UIAbility时，需要提供abilityName和bundleName参数，并使用显式Want方式来启动。关于如何使用显式Want方式启动应用内的UIAbility，请参见[启动应用内的UIAbility](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/uiability-intra-device-interaction)。
