# CPU占用

 #### 应用或元服务后台CPU占用峰值

   **类型**

  规则

   **适用设备**

  手机，平板，智慧屏，车机，手表

   **应用形态适用性**

  鸿蒙应用，鸿蒙元服务

   **说明**

  无
