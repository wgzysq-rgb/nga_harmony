# 是否可以申请长时任务，在游戏前台下载资源包过程中切后台时免冻结并继续下载资源包？

 可以。

 游戏可以申请[dataTransfer类型的长时任务](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/continuous-task)，在游戏前台切后台后，游戏可以保持免冻结并继续下载资源包。
