# 概述

 守护策略管理是Screen Time Guard Kit的核心功能。该模块支持应用添加、修改、查询、删除屏幕时间守护策略，并通过调用启动和停止接口来管控策略生效与否，从而实现对屏幕使用行为的有效监管与优化，帮助用户科学规划屏幕使用时间。
