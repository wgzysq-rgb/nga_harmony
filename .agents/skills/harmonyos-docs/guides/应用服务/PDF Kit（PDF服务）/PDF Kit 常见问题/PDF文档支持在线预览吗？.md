# PDF文档支持在线预览吗？

 PDF Kit不支持在线预览，需要把文档下载到本地，才能预览，具体可参见[预览PDF文档](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/pdf-pdfview-component)。
