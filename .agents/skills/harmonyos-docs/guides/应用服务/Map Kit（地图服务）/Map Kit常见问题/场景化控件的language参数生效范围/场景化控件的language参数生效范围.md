# 场景化控件的language参数生效范围

 
不传language：UI界面、地图、向云侧请求后返回的数据跟随系统语言。

  传language：UI界面和地图跟随系统语言；请求云侧的结果根据language获取，部分数据缺失传参语言默认使用英文兜底。
