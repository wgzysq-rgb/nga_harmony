# Payment Kit的签名公私钥怎么获取？

 签名的公私钥对是由开发者自行生成，生成后将公钥上传到[华为支付商户平台](https://petalpay-merchant.cloud.huawei.com/)后可获取相应的请求参数authId，具体流程查看[准备证书](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/payment-certificates-config#生成商户证书)章节。
