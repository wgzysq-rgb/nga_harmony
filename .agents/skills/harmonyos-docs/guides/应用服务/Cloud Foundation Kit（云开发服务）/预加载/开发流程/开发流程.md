# 开发流程

 开发预加载，总体流程如下：

 

 
[配置预加载](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/cloudfoundation-prefetch-config)：在AGC云端预加载界面选择数据来源类型，并进行相关配置。

  [开发预加载资源接口](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/cloudfoundation-prefetch-cloud-interdev)：根据选择的数据来源类型，开发对应的云侧接口，以提供预加载所需的资源数据。

  [调用预加载](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/cloudfoundation-call-installprefetch)：在工程代码文件中调用预加载。
