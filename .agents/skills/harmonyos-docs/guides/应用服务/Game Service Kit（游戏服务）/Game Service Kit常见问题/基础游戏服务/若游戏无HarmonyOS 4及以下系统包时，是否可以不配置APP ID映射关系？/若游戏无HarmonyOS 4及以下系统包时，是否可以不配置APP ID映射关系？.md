# 若游戏无HarmonyOS 4及以下系统包时，是否可以不配置APP ID映射关系？

 不可以。

 在[配置APP ID映射关系](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/gameservice-gameplayer-huawei#配置app-id映射关系)时，请先创建草稿态的HarmonyOS 4及以下游戏，再与HarmonyOS 5.0及以上游戏进行关联。
