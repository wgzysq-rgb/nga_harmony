# 概述

 从6.0.1(21)版本开始，新增小游戏相关能力。

 小游戏是一种基于行业标准开发的新型免安装应用，无需安装，即点即玩，并且可以添加到设备桌面。开发者一次开发即可将小游戏分发到所有支持行业标准的设备运行。

  #### 约束与限制

小游戏相关能力支持Phone、Tablet、PC/2in1和TV设备。

   #### 使用场景

提供给小游戏运行时应用间共享库调用，以实现小游戏的登录及支付功能。
