# 扫码直达跳转失败

       **问题现象**

    扫码直达跳转失败。

    **解决措施**

    请检查App Linking配置是否正确：

    
           检查开发者网站服务器配置是否正确。

           检查App Linking中网址域名关联是否正确。

           检查应用的module.json5文件中域名关联是否正确。

           检查应用的签名是否正确，参考[手动签名](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-signing#section297715173233)。

    
    详情参考：App Linking的[FAQ](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/app-linking-startupapp#faq)。
