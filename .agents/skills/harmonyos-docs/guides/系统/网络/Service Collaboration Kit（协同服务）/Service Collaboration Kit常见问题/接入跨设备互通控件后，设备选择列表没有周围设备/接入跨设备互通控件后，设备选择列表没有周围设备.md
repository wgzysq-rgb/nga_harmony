# 接入跨设备互通控件后，设备选择列表没有周围设备

 跨设备互通API支持根据特定调用策略调用设备。调用策略：2in1设备可以调用平板和手机，平板可以调用手机，同类型设备不可调用。

 
确保设备调用策略无误。

  设备选择列表仍没有周围设备，请检查双端设备是否开启WLAN、蓝牙，并且登录同一个华为账号。
