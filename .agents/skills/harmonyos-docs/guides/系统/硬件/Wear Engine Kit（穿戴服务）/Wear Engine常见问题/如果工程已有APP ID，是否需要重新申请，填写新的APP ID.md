# 如果工程已有APP ID，是否需要重新申请，填写新的APP ID

 若APP ID已在华为开发者联盟官网完成注册，则无需重复申请。但需确保应用包名、APP ID及证书指纹信息与注册开发者账号时提交的数据完全一致。
