# 加解密介绍

 在数据存储或传输场景中，可以使用加解密操作用于保证数据的机密性，防止敏感数据泄露。

 使用加解密操作时，典型的场景有：

 
使用对称密钥的加解密操作。

  使用非对称密钥的加解密操作。

  使用RSA（PKCS1_OAEP填充模式）时，获取、设置CipherSpecItem参数。
