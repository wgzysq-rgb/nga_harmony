# ohpm --version

 查询 ohpm cli 安装版本。

 #### 命令格式

```
ohpm -v | --version
```
  #### 功能描述

- 打印命令行工具的版本号。
- 可在安装 ohpm 命令行工具之后，对安装版本信息进行校验。
