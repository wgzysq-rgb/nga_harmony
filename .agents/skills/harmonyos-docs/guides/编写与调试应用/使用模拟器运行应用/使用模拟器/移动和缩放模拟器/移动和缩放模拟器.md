# 移动和缩放模拟器

 
移动模拟器您可以使用鼠标拖动模拟器到屏幕的指定位置。首先将鼠标放在屏幕边缘，当鼠标变成样式，按住鼠标左键并移动即可拖动模拟器。当模拟器被拖动到期望位置后，松开鼠标左键即可停止拖动。

 缩放模拟器如需改变模拟器大小，将鼠标放到屏幕四角的任意一处，当鼠标变成，按住鼠标左键并移动即可缩放模拟器。当模拟器被缩放到期望大小后，松开鼠标左键即可完成缩放。
