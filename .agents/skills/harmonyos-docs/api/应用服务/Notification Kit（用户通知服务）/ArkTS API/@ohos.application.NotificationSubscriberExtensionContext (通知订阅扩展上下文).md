@ohos.application.NotificationSubscriberExtensionContext (通知订阅扩展上下文)

NotificationSubscriberExtensionContext模块提供通知订阅者扩展能力。

本模块首批接口从API version 22开始支持。后续版本的新增接口，采用上角标单独标记接口的起始版本。

本模块接口仅可在Stage模型下使用。
