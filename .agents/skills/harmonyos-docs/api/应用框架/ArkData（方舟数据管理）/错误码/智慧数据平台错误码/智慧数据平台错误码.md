31300000 服务内部异常

错误信息

Inner error.

错误描述

当调用智慧数据平台模块相关接口时，若服务内部异常，会报此错误码。

可能原因

服务内部状态异常，例如应用在没有加载对应嵌入模型后调用获取对应嵌入向量的结果，此时属于模型未加载，获取向量化接口就是异常调用。

处理步骤

开发者排查是否按接口文档正确使用接口。

定时重试操作，如间隔1s或者按照指数增长间隔重试，如果依然无法解决，可以提示用户重启应用、升级应用或升级设备版本。
