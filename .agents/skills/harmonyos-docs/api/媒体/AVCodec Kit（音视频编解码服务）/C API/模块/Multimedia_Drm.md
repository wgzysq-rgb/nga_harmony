Multimedia_Drm

概述

该功能使第三方应用程序可以自行实现媒体解封装和解复用功能，而不是使用系统提供的。

在创建DRM实例和会话后，可以调用DRM提供的解密接口进行解密。解密参数结构定义了解密参数的传输格式。
