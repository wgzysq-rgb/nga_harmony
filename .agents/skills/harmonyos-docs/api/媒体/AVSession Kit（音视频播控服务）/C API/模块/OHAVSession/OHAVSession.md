# OHAVSession

                 #### 概述

     提供播控模块C接口定义。

     开发者可使用OHAVSession，作为媒体会话提供方接入AVSession服务，对应的开发指南及样例可参考[媒体会话提供方](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/using-ohavsession-developer)。

     **系统能力：** SystemCapability.Multimedia.AVSession.Core

     **起始版本：** 13
