# input

  #### 概述

提供多模态输入域的C接口。

 **起始版本：** 12

   #### 文件汇总

 名称 描述   [oh_axis_type.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-oh-axis-type-h) 输入设备的轴事件结构和枚举。  [oh_input_manager.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-oh-input-manager-h) 提供事件注入和关键状态查询等功能。  [oh_key_code.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-oh-key-code-h) 按键设备的键值。  [oh_pointer_style.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-oh-pointer-style-h) 鼠标光标的样式。
