32500001 服务异常

错误信息

Service exception.

错误描述

当调用deviceStatus模块on、off接口时，若服务异常，会报此错误码。

可能原因

服务状态异常。

处理步骤

定时重试操作，如间隔1s或者按照指数增长间隔重试。

连续重试3次不可用则停止尝试，期间可优先尝试获取器件列表方式进一步获取设备可用性。
