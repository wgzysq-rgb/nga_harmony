14600101 操作设备失败

错误信息

Device operation failed.

错误描述

当调用vibrator模块startVibration接口时，若hdi服务异常或者设备被占用，会报此错误码。

可能原因

访问hdi服务状态异常。

当前设备被占用。

调用振动时，系统中触觉管理相关开关被关闭。

处理步骤

间隔一段时间重试操作，或按指数增长间隔重试；连续重试3次不可用则停止尝试，期间可优先尝试获取器件列表方式进一步获取设备可用性。

设置振动优先级较低，尝试设置更高优先级。

检查系统触觉开关是否被关闭或三态开关是否处理静音状态，是否正确使用了对应开关管理的usage字段。
