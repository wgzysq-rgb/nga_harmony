  #### 概述

用于描述占位符跨度的结构体。

 **起始版本：** 11

 **相关模块：** [Drawing](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-drawing)

 **所在头文件：** [drawing_text_typography.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-drawing-text-typography-h)

 名称 描述   double width 占位符宽度。  double height 占位符高度。  [OH_Drawing_PlaceholderVerticalAlignment](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-drawing-text-typography-h#oh_drawing_placeholderverticalalignment) alignment 占位符对齐方式。  [OH_Drawing_TextBaseline](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-drawing-text-typography-h#oh_drawing_textbaseline) baseline 占位符基线。  double baselineOffset 占位符基线偏移。
