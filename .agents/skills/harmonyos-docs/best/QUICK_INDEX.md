## 使用方法

1. 根据下方分类表，确定目标最佳实践所属分类
2. 读取对应分类的子索引文件（路径：`best/_index/分类名.md`）
3. 在子索引中查找最佳实践名称，获取文档相对路径
4. 拼接完整路径读取文档：`技能根目录/相对路径`（技能根目录 = SKILL.md 所在目录）

## 快速分类判断

- **性能/丢帧/卡顿/内存/启动时延** → 性能
- **崩溃/Crash/稳定性/日志规范** → 稳定性
- **音频/视频/相机/扫码** → 媒体
- **多设备/折叠屏/平板/智慧屏** → 一次开发，多端部署
- **功耗/省电/后台优化** → 功耗
- **ArkUI/ArkWeb/文件/程序包** → 应用框架
- **布局/弹窗/拖拽/手势** → 布局与弹窗
- **安全/权限/加密** → 应用安全
