```

**Step 2（需要验证码时显示）**：

```
┌─────────────────────────┐
│                         │
│       N G A             │
│                         │
│  ┌─────────────────┐    │
│  │  用户名           │    │

│  └─────────────────┘    │
│  输入图形验证码(6字符)    │
│  ┌──────┐  ┌────────┐   │
│  │ 验证码 │  │ [图片] │   │
│  └──────┘  └────────┘   │
│  换一个                   │
│                         │
│  ┌─────────────────┐    │
│  │      继  续       │    │
│  └─────────────────┘    │
│                         │
└─────────────────────────┘
```

### 数据流

```
Step 1:
  输入用户名密码 → POST /api/auth/login { username, password }
    → 直接成功: 存储 token → navigate('/')
    → 需要验证码: 显示验证码区域

Step 2:
  输入验证码 → POST /api/auth/login { loginToken, captchaId, captcha }
    → 成功: 存储 token → navigate('/')
    → 失败: 显示错误 Toast + 刷新验证码
```

### 关键逻辑

- 验证码**不预加载**，仅在登录返回 `needCaptcha` 后显示
- 验证码图片点击或"换一个"链接可刷新 (`GET /api/auth/captcha?id=xxx`)
- 登录按钮文字：Step 1 "登 录"，Step 2 "继 续"
- 验证码为 **6 字符** (maxlength=6)
- 登录成功后将 token 存入 `appStore.state.token` 和 `localStorage`
- 已登录用户直接跳转首页

```

        → 渲染主题卡片列表

翻页 → navigate('/forum/:fid/page/:page') 或更新参数
```

### 主题类型样式

根据 `type` 位掩码显示标签：
- `1024` (locked) → 🔒 标记
- `8192` (attachment) → 📎 标记

根据 `titleFont` 位掩码着色标题：
- `1` (red) → 红色标题
- `2` (blue) → 蓝色标题
- `32` (bold) → 加粗

```

### 楼层组件 `<nga-post-floor>`

这是最复杂的 UI 组件，需要处理：

| 功能 | 实现 |
|------|------|
| 帖子内容 | `innerHTML` 渲染已解析 HTML |
| 附件图片 | 点击放大（lightbox 效果），剥离缩略图后缀加载原图 |
| 用户头像 | `<nga-user-avatar>` 组件 |
| 引用跳转 | `pid` 定位 + 跨页导航 |
| 评论展开/收起 | 默认折叠，点击展开 |
| 快速回复 | 底部固定输入栏 |
| 楼层号 | `lou` 字段，0 = 楼主 |
| 黑名单用户 | 显示 "该用户已被屏蔽" |
| 被禁言用户 | 灰色标记 |
| 评分 | `score` / `score_2` |
| 客户端来源 | `fromClient` 字段，显示为灰色标签（如 "来自 Android"） |
| 匿名用户 | 解码后的中文用户名 + 匿名标识 |

### 图片查看器

点击帖子中的图片时全屏查看：

```typescript
// 简单 lightbox 实现
document.addEventListener('click', (e) => {
  const img = (e.target as HTMLElement).closest('.nga-img, .nga-attach-img');
  if (img) {
    const overlay = document.createElement('div');
    overlay.className = 'lightbox';
    overlay.innerHTML = `<img src="${(img as HTMLImageElement).src}">`;
    overlay.addEventListener('click', () => overlay.remove());
    document.body.appendChild(overlay);
  }
});
```

### 关键交互

- 域名切换后立即更新后端 `NgaClient.setDomain()`（通过 `POST /api/settings/domain`）
- 暗色/亮色主题切换修改 `document.documentElement.dataset.theme`
- 字号切换修改 CSS 变量 `--font-size-base`
- 多账号切换通过更换 `appStore.state.token` 实现
- 黑名单存储在 localStorage，修改后同步到 `appStore.state.blackList`
